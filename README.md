# Jogo da Forca

Projeto desenvolvido em Python para praticar lógica de programação e manipulação de strings.

## Descrição

O jogo escolhe uma palavra e o jogador deve tentar adivinhar suas letras antes de atingir o limite de erros.

## Conceitos praticados

- Loops (`while`)
- Condicionais (`if`, `elif`, `else`)
- Manipulação de strings
- Listas
- Entrada de dados com `input()`
- Controle de fluxo
- Lógica de programação

## Funcionalidades

- Escolha de palavra secreta
- Tentativas do jogador
- Contagem de erros
- Exibição do progresso da palavra
- Mensagens de vitória e derrota

## Objetivo do projeto

Este projeto foi desenvolvido como parte dos meus estudos em Python para praticar lógica de programação e resolução de problemas.
